# Veri Setleri

Bu projede üç farklı binary classification veri seti kullanılmıştır.

## IBM HR Analytics Employee Attrition & Performance

- Domain: İnsan kaynakları
- Amaç: Çalışanın şirketten ayrılıp ayrılmayacağını tahmin etmek
- Hedef sütun: `Attrition`
- Kaynak: Kaggle — `pavansubhasht/ibm-hr-analytics-attrition-dataset`

## Telco Customer Churn

- Domain: Telekomünikasyon
- Amaç: Müşterinin aboneliğini sonlandırıp sonlandırmayacağını tahmin etmek
- Hedef sütun: `Churn`
- Kaynak: Kaggle — `blastchar/telco-customer-churn`

## Mushroom Classification

- Domain: Biyoloji
- Amaç: Mantarın yenilebilir veya zehirli olduğunu tahmin etmek
- Hedef sütun: `class`
- Kaynak: Kaggle — `uciml/mushroom-classification`
